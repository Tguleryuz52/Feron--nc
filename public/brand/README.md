# ✨ Kılavuz: Marka (Brand) Varlıklarını Buraya Ekle

Sekme ikonu (Favicon), alternatif logo tasarımları veya global sitede kullanılabilecek marka unsurlarını buraya yüklemelisin.

**Gereken Dosyalar:**

- `favicon.ico` (Sitenin sekmesinde gözükecek küçük ikon)
- `logo-dark.svg` (Siyah zeminler için açık renk logo - eğer kullanacaksan)
- `logo-light.svg` (Beyaz zeminler için siyah logo - eğer kullanacaksan)
  _(Not: Eğer logoyu SVG olarak elinde tutmuyorsan sorun değil, Opus 'Satoshi Serif' fontunu kullanarak metin tabanlı "feron." logosunu kodla da birebir çizebilir.)_

**Ne Yapmalısın?**
Markanla ilgili spesifik ikonik materyalleri bu `public/brand` klasörünün içine yapıştır. Opus `favicon` okumalarını ve özel marka assetlerini buradan çekecektir.
